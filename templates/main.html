Preciso: Effictive Content Improvisation System

Preciso is a content improvisation system. It includes three functionalities: Spell Checking, Tone Checking and Summarization. Spell Checking is a mechanism where Preciso will point out any erroneous word after parsing the entire document. If any such word is found, then Preciso will suggest that mistake through the correction panel. Tone Checking tells the user about the context of the document whether it is positive, negative or neutral. Further, it is not always possible to read the entire document in this busy world. Also before reading one, everyone wants to know what the content exactly deals with. Summarization provides one such provision to display a window of a document. Preciso is implemented in python, HTML, CSS and JavaScript. This project was sponsored by Persistent Systems Limited.

Infinity Motors: Car Showroom Management System.

Infinity Motors is a windows-based desktop application created to store the information of all cars and customers. It of cars and customers is stored in MongoDB database and C# .Net is used for frontend.
